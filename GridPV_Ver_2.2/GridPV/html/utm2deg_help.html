utm2deg

Function to convert vectors of UTM coordinates into Lat/Lon vectors (WGS84).

Contents

  • Syntax
  • Description
  • Inputs
  • Outputs
  • Copyright

Syntax

[Lat,Lon] = utm2deg(xx,yy,utmzone)

Description

Function to convert vectors of UTM coordinates into Lat/Lon vectors (WGS84). Some code has been extracted from UTMIP.m function by Gabriel Ruiz Martinez. Author: Rafael Palacios, Universidad Pontificia Comillas, Madrid, Spain

Inputs

  • x - x coordinate in UTM
  • y - y coordinate in UTM
  • utmzone - the UTM zone designation

Outputs

  • Lat - Latitude vector. Degrees. +ddd.ddddd WGS84
  • Lon - Longitude vector. Degrees. +ddd.ddddd WGS84

Example

Calculate the latitude/longitude for 6 example locations in difference UTM zones

x=[ 458731;  407653;  239027;  230253;  343898;  362850];
y=[4462881; 5126290; 4163083; 3171843; 4302285; 2772478];
utmzone=['30 T'; '32 T'; '11 S'; '28 R'; '15 S'; '51 R'];
[Lat, Lon]=utm2deg(x,y,utmzone);
[Lat';Lon']

ans =

   40.3154   46.2839   37.5778   28.6456   38.8556   25.0618
   -3.4857    7.8012 -119.9552  -17.7595  -94.7990  121.6403
